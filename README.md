# Mini-App - User Registration and Authentication

## Project Overview
This project is a centralized Authentication and Profile Management module. It acts as a gatekeeper between the public internet and "Protected Pages" by managing user state through session tokens.

## Purpose
To familiarize and design a system flow for User Registration, Login, and Logout.

## Tech Stack
* **Frontend:** React (Web Client)
* **Backend:** Spring Boot (Java) 
* **Database:** MySQL (Relational Database) 
* **Security:** BCrypt/Argon2 for password hashing
